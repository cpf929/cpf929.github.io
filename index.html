linux相关

  • 修改RedHat的yum源
    下载163或者阿里的yum配置文件

    wget http://mirrors.163.com/.help/CentOS6-Base-163.repo
    替换CentOS-Base-163.repo中 $releaserver 参数为对应的centos版本，如6， 使用vi 的替换功能.
    :%s/$releaserver/6/g
    yum makecache
    yum install yum-fastestmirror 安装fastestmirror插件可自动选择最快的yum 源
